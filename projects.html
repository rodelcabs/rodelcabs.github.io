My Projects

Here are the projects I've been involved to, some of it was the main project I've worked on. These projects were based on the requirements and needs by different clients.

Project Management Tool
This project was designed to manage projects, tasks, incidents, and even productivity. It has a “Kanban” and other team management features. The project was built in JavaScript using NodeJS as its runtime environment, and ExpressJS as its framework. We used Sequelize for data management through MySQL. On the client-side we used EJS as template engine and other libraries as well as Bootstrap and jQuery.
File Duplicate Manager Tool (Automation)
This project was designed to find duplicates on uploaded files and creates new script file for client. The project was built in PHP using CodeIgniter framework and MySQL as the backend, it also had some native PHP scripts running as background service for its automation purposes while HTML, CSS, and JavaScript was used on client-side. We also used different plugins and libraries as well as Bootstrap and jQuery.
Change Request Management Tool (Automation)
This project was designed to automate the change requests, scheduling, and broadcast advisories. The project was built in PHP using CodeIgniter framework and MySQL as the backend while HTML, CSS, and JavaScript is used on client-side. We also used different plugins and libraries as well as Bootstrap and jQuery.
Key Logger Tool (with Bar Code Scanner)
This project was designed to log the key’s borrowing history, scheduling. It has a feature of barcode scanner using the camera available to device. Each key has its unique bar code and this helps the process of borrowing the keys more time-efficient for easier logging. The project was built in PHP using CodeIgniter framework and MySQL as the backend while HTML, CSS, and JavaScript is used on client-side. We also used different plugins and libraries as well as Bootstrap, jQuery, and QuaggaJS for the app to be able to scan bar codes.
Examination Tool
This project for clients to take their exams, and eventually get their certificate. The project was built in PHP using CodeIgniter framework and MySQL as the backend while HTML, CSS, and JavaScript is used on client-side. We also used different plugins and libraries as well as Bootstrap and jQuery
Customer Satisfactory Tool (Survey)
This project is used to measure the performance or service of teams, it gathers feedback from team members and gives an analysis and reports from gathered data. This project is developed with CodeIgniter Framework (PHP) and MySQL for the server-side, and HTML, CSS, JavaScript on the client-side.
Proactive Monitoring Tool
This project is used to monitor IP/Port if they were active or not. This project is developed with CodeIgniter Framework (PHP) and MySQL for the server-side, it also had a script written in Python and running as background service, while HTML, CSS, JavaScript on the client-side.
